# Setup

Flutter 3.x

Packages:
```
flutter_riverpod

# Environment Setup

Copy:
```
cp .env.example .env
```

Fill values:
```
SUPABASE_URL=
SUPABASE_ANON_KEY=
```

Install:
```
flutter pub get
```

Run:
```
flutter run
